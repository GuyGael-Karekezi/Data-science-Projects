# Project 5: Financial Time Series and PCA

## Overview
This project analyzes stock time-series data, applies PCA, and compares forecasting approaches.

## Main Work
- Download and preprocess DJIA stock data
- Compute returns and correlation patterns
- Apply PCA and interpret explained variance
- Time-series diagnostics (stationarity, ACF/PACF, decomposition)
- Forecasting with SMA, ARIMA, and LSTM

## Project Structure
- `notebooks/project5_financial_time_series_pca_forecasting.ipynb`: full workflow
- `reports/`: report files removed from this repository

## How to Run
1. Open the notebook in Jupyter.
2. Install required Python libraries used in the notebook.
3. Run cells from top to bottom.
