# Project 3: Loan Status Prediction

## Overview
This project builds classification models to predict loan approval status.

## Main Work
- Exploratory data analysis and data quality checks
- Missing value and outlier handling
- Categorical encoding and feature engineering
- Model training and comparison (Logistic Regression, SVM, SGD)
- Evaluation using confusion matrix, ROC, and AUC

## Project Structure
- `notebooks/project3_loan_status_prediction.ipynb`: full workflow
- `reports/`: report files removed from this repository

## How to Run
1. Open the notebook in Jupyter.
2. Install required Python libraries used in the notebook.
3. Run cells from top to bottom.
